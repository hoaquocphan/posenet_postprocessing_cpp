## Postprocessing of PoseNet C++

1/ clone python code [hoaquocphan/posenet-python](https://github.com/hoaquocphan/posenet-python)
2/ run python code
./run.sh
3/ copy output_data folder from python code and place at posenet_postprocessing folder
4/ prepare image data as defined in posenet_postprocessing/output_data/image_name.txt
5/ run postprocessing
make
./postprocessing
